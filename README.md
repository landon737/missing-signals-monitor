# Missing Signals Monitor – Live Prototype

A Streamlit dashboard that visualises BTC price, liquidity, stablecoin peg stability,
and a composite risk score in one view.

## Included features (current)
- Simulated BTC price feed with event-triggered shock
- Simulated Global Liquidity Index
- Stablecoin Peg Monitor (USDC)
- Composite Risk Scoring Panel
- Visual event marker

## Future enhancements
- Live BTC/ETH price data
- Real liquidity data (TGA, RRP, Central Bank balance sheets)
- Stablecoin peg from Binance/Curve
- Exchange fragility indicators
- On-chain staking/bridging/smart-money flows
- Narrative velocity scanning
- Real-time alerts
- Prediction engine for ±8% moves
